# 利用多线程解决 N 皇后问题

## ✏️项目介绍

- n 皇后问题研究的是如何将 n 个皇后放置在 n×n 的棋盘上，并且使皇后彼此之间不能相互攻击，是回溯算法的典型案例
- 由于算法的时间复杂度过高[O(n!)]，导致 n 值过高时，运行特别耗时。一般我们都是采用单线程来解决此类问题，因此我们希望发挥多核 CPU 的优势，考虑使用多线程解决 N 皇后问题，提高运行的效率
